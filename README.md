# QRCODE COMMERCE
- Criando um Gerador de QR Codes para E-commerces 

## STACKS
- Nodejs
- Chalk
- Qrcode-terminal
- Prompt

## FEATURES

## Como utilizar 
- Clone o repositório
- Dentro do projeto, digite o comando para instalar as dependências:
```
npm i
```
- Rode a aplicação com o comando:
```
npm run dev
```
